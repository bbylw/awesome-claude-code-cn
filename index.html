斜杠命令

斜杠命令是定制化的、经过精心优化的提示，用于控制 Claude 的行为以执行特定任务。

版本控制

/commit

使用适当的表情符号按照常规提交格式创建 git 提交，遵循项目标准并创建描述性消息。

版本控制

/create-pr

通过处理整个工作流程简化拉取请求创建：创建新分支、提交更改、格式化修改的文件并提交 PR。

代码分析

/tdd

使用测试驱动开发原则指导开发，强制执行 Red-Green-Refactor 纪律，与 git 工作流程集成。

上下文加载

/context-prime

通过加载仓库结构、设置开发上下文、建立项目目标和定义协作参数，用全面的项目理解预热 Claude。

文档

/create-docs

分析代码结构和目的以创建全面的文档，详细说明输入/输出、行为、用户交互流程。
